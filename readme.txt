Output Files
    result.txt
        Stores all results from Run 0 to Run 9999 (10000).
        Each line is in the format (A, B, C, D)
            A - Run number
            B - Initially infected population on Day 0
            C - Population infected after 30 days (excluding initial)
            D - Population dead after 30 days

    infected.txt / dead.txt / initial.txt
        Stores the mean number of infected/dead/initial nodes after 30 days (excluding initial) on first line
        Stores the standard deviation of infected/dead/initial nodes after 30 days (excluding initial) on second line.
